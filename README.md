# micro-app-example

qiankun 微前端 demo，父应用 Umi，子应用使用 `react`, `vue` 。

主应用：https://gis.giscafer.com/
## 开始

进入到各子应用目录下执行：

安装工程依赖

```
npm i
```


一键启动应用
```
npm start
```

<!-- 通过 [http://localhost:8080/](http://localhost:8080/) 访问主应用。 -->

## 发布

```
npm run build
```
